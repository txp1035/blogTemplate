## Hexo 常用命令

### 使用方法

`hexo <command> [options]`

### 命令\<command\>

\<clean\>:删除生成的文件和缓存。

\<config\>:获取或设置配置。

\<deploy\>:部署你的网站。

\<generate\>:生成静态文件。

\<help\>:获取命令的帮助。

\<init\>:创建一个新的 HEXO 文件夹。

\<list\>:列出网站信息。

\<migrate\>:将你的网站从其他系统迁移到 HEXO。

\<new\>:创建一个新的发布的页面。

\<publish\>:将草案从草稿移动到发布文件夹。

\<render\>:在档案与渲染器插件。

\<server\>:启动服务器。

\<version\>:显示版本信息。

### 全局选项\[options\]

\[--config\]:指定配置文件，而不是使用\_config.yml

\[--cwd\]:指定 CWD

\[--debug\]:在终端中显示所有冗长的消息

\[--draft\]:展示稿件

\[--safe\]:禁用所有插件和脚本

\[--silent\]:在控制台隐藏输出
